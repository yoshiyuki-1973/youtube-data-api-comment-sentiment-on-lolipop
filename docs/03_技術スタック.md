# 技術スタック

YouTube動画コメント感情分析システム（PHP / ロリポップ版）

## 1. 概要

本ドキュメントでは、PHP / ロリポップ版で使用する技術要素の一覧とバージョン情報を記載する。

---

## 2. 言語・ランタイム

| 技術 | バージョン | 用途 |
|------|-----------|------|
| PHP | 8.1以上 | アプリケーション言語 |

---

## 3. フレームワーク・ライブラリ

### 3.1 Web UI

| 技術 | バージョン | 用途 | 提供方法 |
|------|-----------|------|---------|
| Chart.js | 4.4.0 | 感情分析結果のドーナツグラフ | CDN |
| HTML / CSS / JavaScript | - | UI実装 | 自前実装 |

### 3.2 AI 感情分析

| 技術 | バージョン | 用途 |
|------|-----------|------|
| Grok API（xAI） | grok-3-mini | 感情分析（ポジ/ネガ/ニュートラル） |

### 3.3 API クライアント

| 技術 | バージョン | 用途 |
|------|-----------|------|
| YouTube Data API v3 | - | 動画メタデータ・コメント取得 |

PHPの標準拡張（`curl`、`json`）のみで実装。外部ライブラリ不要。

### 3.4 データベース

| 技術 | バージョン | 用途 |
|------|-----------|------|
| MySQL | 5.7以上 | 分析結果キャッシュ（24時間） |
| PDO | PHP標準 | MySQLアクセス |

### 3.5 テスト

| ライブラリ | バージョン | 用途 |
|-----------|-----------|------|
| PHPUnit | 10.x | テストフレームワーク |

### 3.6 パッケージ管理

| ツール | バージョン | 用途 |
|--------|-----------|------|
| Composer | 最新版 | PHPUnit等の依存管理 |

## 4. 外部サービス

| サービス | 用途 | 制限・注意事項 |
|---------|------|--------------|
| YouTube Data API v3 | 動画メタデータ・コメント取得 | 10,000ユニット/日（無料枠） |
| Grok API（xAI） | AIによる感情分析 | 無料枠あり（レート制限に注意） |

### 4.1 Grok API 設定値

| 項目 | 値 |
|------|-----|
| エンドポイント | `https://api.x.ai/v1/chat/completions` |
| デフォルトモデル | `grok-3-mini` |
| 変更可能モデル | `grok-beta` / `grok-2-1212` / `grok-3-mini` |
| タイムアウト | 60秒 |
| バッチサイズ | 10件/リクエスト |
| 設定キー | `config.php` の `GROK_API_KEY` / `GROK_MODEL` |

### 4.2 YouTube Data API 設定値

| 項目 | 値 |
|------|-----|
| エンドポイント | `https://www.googleapis.com/youtube/v3` |
| 使用メソッド | `videos.list`、`commentThreads.list` |
| タイムアウト | 30秒 |
| 設定キー | `config.php` の `YOUTUBE_API_KEY` |
