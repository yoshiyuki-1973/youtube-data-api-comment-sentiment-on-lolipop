# 操作マニュアル

YouTube動画コメント感情分析システム ユーザー操作ガイド

## 1. 概要

本システムは、YouTube動画のコメントを取得し、Gemini AI（Google）による感情分析を行い、結果をブラウザで可視化するWebアプリケーションである。

| 項目 | 内容 |
|------|------|
| アクセス方法 | ブラウザでシステムURLにアクセス |
| 動作環境 | 最新のChrome / Firefox / Safari / Edge |
| 認証 | 不要（誰でもアクセス可能） |

---

## 2. 分析の実行

### 2.1 URLまたは動画IDを入力する

画面上部の入力欄に、以下のいずれかの形式でYouTube動画を指定する:

| 入力形式 | 例 |
|---------|-----|
| 動画ID（11文字） | `dQw4w9WgXcQ` |
| 通常URL | `https://www.youtube.com/watch?v=dQw4w9WgXcQ` |
| 短縮URL | `https://youtu.be/dQw4w9WgXcQ` |
| Shorts URL | `https://www.youtube.com/shorts/dQw4w9WgXcQ` |
| 埋め込みURL | `https://www.youtube.com/embed/dQw4w9WgXcQ` |

### 2.2 コメント取得件数を選択する

「コメント取得件数」のセレクトボックスで、分析するコメントの件数を選択する:

| 選択肢 | 用途 |
|--------|------|
| 10件 | 素早く確認したい場合 |
| 30件 | 通常利用（推奨） |
| 50件 | より多くのコメントを分析 |
| 100件 | 全体の傾向を把握したい場合 |

> **注意**: 件数が多いほど分析時間が長くなる（100件で最大約50秒）。

### 2.3 分析を開始する

「**分析開始**」ボタンをクリックする。

- 分析中はスピナーが表示される
- 同じ動画を24時間以内に再分析した場合はキャッシュから即座に返る
- API 呼び出し中は画面を閉じないこと

## 3. 結果の見方

### 3.1 動画情報

分析した動画の基本情報が表示される:

| 表示項目 | 説明 |
|---------|------|
| 動画タイトル | 動画のタイトル |
| チャンネル名 | 投稿者のチャンネル名 |
| 再生回数 | 動画の総再生回数 |
| 高評価数 | 動画の高評価数 |
| コメント数 | 動画の総コメント数 |
| 公開日 | 動画の公開日 |

### 3.2 感情分析グラフ

ドーナツグラフで全コメントの感情分布が表示される:

| 色 | 感情 | 説明 |
|----|------|------|
| 緑（#34d399） | ポジティブ | 肯定的なコメントの割合 |
| 赤（#f87171） | ネガティブ | 否定的なコメントの割合 |
| グレー（#9ca3af） | ニュートラル | 中立・判定困難なコメントの割合 |

グラフの下に以下の数値も表示される:

| 項目 | 説明 |
|------|------|
| 分析件数 | 分析したコメントの総数 |
| ポジティブ件数 / 比率 | 肯定的と判定されたコメント数と割合 |
| ネガティブ件数 / 比率 | 否定的と判定されたコメント数と割合 |
| ニュートラル件数 / 比率 | 中立と判定されたコメント数と割合 |
| 平均スコア | 全コメントの平均感情スコア（0.0〜1.0） |

### 3.3 コメント一覧

感情分析済みのコメントが一覧表示される。

#### フィルタタブ

| タブ | 表示内容 |
|------|---------|
| すべて | 全コメント |
| ポジ | ポジティブと判定されたコメント |
| ネガ | ネガティブと判定されたコメント |
| ニュートラル | ニュートラルと判定されたコメント |

#### 各コメントカードの表示内容

```
[ポジ] 最高の動画でした！また見たい。            ← コメント本文
P:0.85 N:0.10 Neu:0.05                         ← 感情スコア
投稿者: ユーザー名  👍 42  2025-01-01           ← 投稿者・いいね数・日時
```

| 表示 | 説明 |
|------|------|
| `[ポジ]` / `[ネガ]` / `[ニュートラル]` | 感情ラベル（最大スコアのクラス） |
| `P:0.85` | ポジティブスコア（0.00〜1.00） |
| `N:0.10` | ネガティブスコア（0.00〜1.00） |
| `Neu:0.05` | ニュートラルスコア（0.00〜1.00） |
| 👍 数字 | コメントのいいね数 |

## 4. スコアの読み方

### 4.1 感情スコアとは

各コメントに対して、ポジティブ・ネガティブ・ニュートラルの3つのスコアが算出される。3つのスコアの合計は常に **1.0** になる。

| スコア | 意味 | 高スコアの例 |
|--------|------|------------|
| ポジティブ（P） | 肯定的な感情の強さ | 「最高！面白かった！また来ます」 |
| ネガティブ（N） | 否定的な感情の強さ | 「つまらない、時間の無駄だった」 |
| ニュートラル（Neu） | 中立・判定困難の強さ | 「動画を見ました」「なるほど」 |

### 4.2 感情ラベルの判定基準

3つのスコアのうち最も高いものが感情ラベルになる:

| 条件 | ラベル |
|------|--------|
| ポジティブスコアが最大 | **ポジ** |
| ネガティブスコアが最大 | **ネガ** |
| ニュートラルスコアが最大 | **ニュートラル** |

---

## 5. キャッシュについて

同じ動画ID × 同じコメント件数で **24時間以内** に再度分析を実行した場合、YouTube API および Gemini API を呼び出さずにキャッシュから結果を返す。

- キャッシュヒット時は非常に高速に結果が返る
- キャッシュを強制的に更新したい場合は、コメント件数を変更して再分析するか、管理者に依頼する
- 同じコメント件数のままでは、24時間以内に手動で再分析しても直前のキャッシュ結果が優先される

---

## 6. エラーメッセージと対処法

| エラーメッセージ | 原因 | 対処法 |
|----------------|------|--------|
| 「YouTube APIの認証に失敗しました」 | APIキーが無効 | 管理者に連絡 |
| 「YouTube APIのクォータを超過しました」 | 1日の上限（10,000ユニット）超過 | 翌日（日本時間17:00〜18:00リセット）に再試行 |
| 「動画が見つかりません」 | 動画IDが誤り、または非公開・削除済み | URLを再確認する |
| 「この動画はコメントが無効化されています」 | 動画のコメント機能が無効 | 別の動画で試す |
| 「Gemini APIへの接続に失敗しました」 | ネットワークエラーまたはAPIキー不正 | 時間をおいて再試行 |
| 「しばらく待ってから再試行してください」 | Gemini API のレート制限 | 数分後に再試行 |
| 「有効なYouTube動画IDまたはURLを入力してください」 | URL形式が不正 | 正しいURL形式で入力し直す |
