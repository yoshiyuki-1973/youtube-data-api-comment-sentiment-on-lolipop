# 企画書

YouTube動画コメント感情分析システム（PHP / ロリポップ版）

## 1. プロジェクト概要

YouTube動画のコメントを自動収集し、Gemini AI（Google）による感情分析を行い、コメントのポジティブ・ネガティブ傾向をブラウザで可視化するWebアプリケーションである。分析結果は MySQL に24時間キャッシュし、再アクセス時の API コスト・応答速度を最適化する。

## 2. 背景と目的

### 2.1 背景

YouTubeにおいて、動画のコメント欄はクリエイターと視聴者の重要なコミュニケーションチャネルである。しかし、コメント数が多い動画では全コメントを手作業で確認・分析することが困難であり、コメント全体の感情傾向を定量的に把握する手段が求められている。

### 2.2 目的

- YouTube動画のコメントに対する感情分析を自動化する
- LLM（Gemini API）を活用した日本語コメントへの高精度な感情分析を実現する
- 分析結果をわかりやすいドーナツグラフ付きダッシュボードで可視化する
- ロリポップ共有サーバー（月500円）で稼働するコスト効率の高いシステムを実現する
- ポートフォリオ作品として、PHP・外部AI API連携・MySQL・フロントエンドを組み合わせたWebシステムを構築する

## 3. ターゲットユーザー

| ユーザー | 用途 |
|---------|------|
| YouTubeクリエイター | 自身の動画に対する視聴者の反応を定量的に把握 |
| マーケティング担当者 | 特定の動画やキャンペーンに対する視聴者感情の分析 |
| データアナリスト | YouTube上のトレンドや感情傾向の調査 |
| ポートフォリオ閲覧者 | 技術力・設計力の評価 |

## 4. 主要機能

### 4.1 コメント収集

- YouTube Data API v3を使用した動画メタデータ・コメントの自動取得
- 複数のYouTube URL形式に対応（通常URL、短縮URL、Shorts、埋め込み、直接ID）
- いいね数順でのコメントソート

### 4.2 感情分析

- Gemini API（Google `gemini-2.5-flash`）によるAI感情分析
- 3値分類（ポジティブ / ネガティブ / ニュートラル）
- 20件ずつバッチ処理でAPIコール数を最小化
- 各コメントに0.0〜1.0のスコアを付与し、合計が1.0になるよう正規化

### 4.3 可視化ダッシュボード

- Chart.js 4.4.0 によるドーナツグラフで感情分布を可視化
- 動画メタデータ（再生回数、高評価数等）の表示
- 感情フィルタタブ付きコメント一覧（すべて / ポジ / ネガ / ニュートラル）

### 4.4 24時間キャッシュ

- MySQL に分析結果を保存し、同一動画ID × コメント件数で24時間以内の再アクセス時は API 呼び出しなしに即座に返却
- YouTube API クォータ（10,000ユニット/日）および Gemini API の利用コストを節約

| 項目 | 技術 |

## 6. システム構成概要

```
ユーザー（ブラウザ）
  │
  ├── GET  index.php
  │     └── HTML + Chart.js フォーム
  │
  └── POST api/analyze.php（AJAX）
        │
        ├── MySQLキャッシュ確認
        │   └── キャッシュあり → 即返却（API呼び出しなし）
        │
        └── キャッシュなし
              ├── YouTube Data API v3 → 動画情報・コメント取得
              ├── Gemini API → 感情分析（20件/バッチ）
              ├── MySQL → 結果保存（24時間キャッシュ）
              └── JSON レスポンス → Chart.jsグラフ表示
```

## 8. 制約事項

- YouTube Data API v3 のクォータ制限（1日10,000ユニット）
- Gemini API の無料枠・レート制限（レート超過時は数分待機が必要）
- ロリポップ共有サーバーのため、長時間処理（100件分析で最大約50秒）に対してタイムアウトの考慮が必要
- 手動実行方式（スケジューラ不使用）
